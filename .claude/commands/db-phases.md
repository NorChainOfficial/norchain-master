# Dashboard Phases Command

Get all phases with their deliverables and progress.

## Instructions

Use the `get_phases` MCP tool:

```
get_phases
```

Present the results in a formatted view showing for each phase:
- Phase ID and name
- Focus area
- Status (pending, active, complete)
- Progress percentage
- Deliverables with their status

Highlight the currently active phase.
