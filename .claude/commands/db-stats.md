# Dashboard Stats Command

Get current NorChain PM Dashboard statistics.

## Instructions

Use the `get_dashboard_stats` MCP tool to retrieve current statistics:

```
get_dashboard_stats
```

Then present the results in a formatted summary showing:
- Current active phase and progress
- Overall project progress
- Task breakdown by status (backlog, in_progress, review, done)
- Compliance status
- Repository count

Format the output as a clean dashboard summary for the user.
