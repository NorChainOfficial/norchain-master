# NorChain Development Skill

Comprehensive development skill for the NorChain blockchain ecosystem.

## Skill Identity

This skill provides expertise in:
- NorChain blockchain development patterns
- Token implementation (NOR, PM-EQ, NV-EQ)
- SmartPay payment processing
- Compliance-aware development

## Activation Triggers

Activate this skill when:
- Working on any NorChain repository
- Implementing blockchain features
- Developing smart contracts
- Building payment flows
- Creating wallet functionality

## Core Knowledge

### Token Classification

| Token | Type | Tradability | Compliance Level |
|-------|------|-------------|------------------|
| NOR | Utility | Public exchanges | Minimal |
| PM-EQ | Security | Private only | Full KYC/AML |
| NV-EQ | Security | Private only | Full KYC/AML |

### Development Phases

1. **Phase 1**: Blockchain Core (Go, PoSA)
2. **Phase 2**: Explorer (NestJS, Next.js)
3. **Phase 3**: Smart Contracts (Solidity)
4. **Phase 4**: Wallets (Swift, Kotlin, Web)
5. **Phase 5**: SmartPay (Payment Processing)
6. **Phase 6**: RWA Portals
7. **Phase 7**: Admin Dashboard
8. **Phase 8**: Coinbase Integration
9. **Phase 9**: Documentation
10. **Phase 10**: Compliance

### Critical Rules

1. **NorChain NEVER handles FIAT** - Always use on-ramp partners
2. **Security tokens are NOT publicly tradable**
3. **All PM-EQ/NV-EQ transfers require KYC verification**
4. **Private secondary market only for security tokens**
